# Car Sales & Service Dashboard (IBM Cognos Analytics)

## Project Overview
This project analyzes sales and service performance of a car dealership using IBM Cognos Analytics.

The dashboard helps regional managers understand:
- Profit trends
- Quantity sold
- Model performance
- Customer sentiment
- Recall patterns

## Tools Used
- IBM Cognos Analytics
- Excel
- Data Visualization

## Dashboards Created

### Sales Dashboard
KPIs included:
- Profit
- Quantity Sold
- Quantity Sold by Model
- Average Quantity Sold
- Profit by Dealer ID

### Service Dashboard
Visualizations included:
- Recalls per Model
- Customer Sentiment Analysis
- Monthly Quantity vs Profit
- Recalls by Model and System

## Key Insights
- Hudson model recorded highest sales volume
- Positive customer sentiment dominated reviews
- Profit peaks during mid-year months
